Proyecto Cuestionario    

Juego de selección simple hecho en JavaScript usando la API fetch para consumir una API pública de un sitio de juegos de preguntas y respuestas. En el juego, el usuario ve 3 opciones para cada pregunta, hay 3 preguntas y la puntuación se marca en la parte superior a medida que avanza el juego.

Tecnologias Utilizadas    

ECMAScript -- JavaScript -- CSS3 -- HTML5
Implementación de objeto Math.
